## Process Management 
- `top          : its shows all process used to monitor & troubleshoot`
- `htop         : you can horizontally & vertically scroll all process`
- `ps -a        : is used to display all process ` 
- `pstree       : it will showing running process in tree format`
- `watch top    : it will monitor top cmd every 2s `
- `kill -9 PID  : forcefully terminate a process`

## File System 
- `df -h    : it shows disk usage`
- `du -h    : it shows directory usage`
- `ls       : list files `
- `pwd      : present/print working directory`
- `cd       : change directory`
- `tree     : it will show current directory structure in the form of tree`    
- `touch    : to create a empty file`
- `cat      : to read a file content`
- `mkdir    : to create a directory`
- `rm -r    : to delete directory`
- `cp       : to copy from file1 to file2`
- `mv       : to move from one directory to other or file1 to file2 or to rename file `

## Networking troubleshooting
- `ping         : used to cheked host is reachable or not`
- `hostname     : to check a server ip/hostname`
- `ip addr      : to check ip address of server`
- `netstat -tuln: to check listening port `
- `dig          : to check dns info `
- `curl -X Get url  : to call an get api `
